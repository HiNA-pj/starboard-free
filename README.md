# Starboard Free

OBS Studio で使えるリアルタイム勝敗カウンターです。

操作画面で WIN / LOSE をカウントすると、OBS のブラウザソースにリアルタイムで反映されます。

---

## 必要環境

- [Node.js](https://nodejs.org/) がインストールされていること

---

## 起動方法

ターミナルを開き、`starboard-free` ディレクトリで以下のコマンドを **1回** 実行するだけです。

```bash
npm run dev:all
```

初回起動時は依存パッケージのダウンロードが自動で行われます。  
その後、API サーバーと画面表示サーバーが同時に起動します。

---

## 各画面の URL

### 操作画面（配信者がクリックしてカウントする画面）

http://localhost:5173/

### OBS overlay 画面（OBS にブラウザソースとして追加する画面）

http://localhost:5173/overlay

---

## 画面の見かた（操作画面）

- **ヘッダー**: 左側にアプリ名、右側にサーバー接続ステータス（「サーバー同期中」または「ローカル動作中」）を表示します。
- **OBS設定パネル**: 画面下部にOBS用URL・URLコピーボタン・推奨サイズ `幅 800 / 高さ 200` を表示します。
- サーバー未接続時も WIN / LOSE カウントはローカルで動作します。

---

## 使い方（操作画面）

| 操作 | 方法 |
|---|---|
| WIN を +1 する | 「WIN」枠内の **+** ボタンをクリック |
| WIN を -1 する | 「WIN」枠内の **−** ボタンをクリック |
| LOSE を +1 する | 「LOSE」枠内の **+** ボタンをクリック |
| LOSE を -1 する | 「LOSE」枠内の **−** ボタンをクリック |
| タイトルを変更する | 上部のテキストボックスに任意のタイトルを入力 |
| スコアをリセットする | 下部の「スコアをリセット」ボタンをクリック |

---

## OBS Studio への設定方法

1. OBS Studio を開きます。
2. ソース一覧の「＋」ボタンをクリックし、「ブラウザ」を選択します。
3. 新しいソースを作成（任意の名前を付けて OK）。
4. 以下の設定を入力します。

| 項目 | 値 |
|---|---|
| URL | `http://localhost:5173/overlay` |
| 幅 | `800` |
| 高さ | `200` |

5. 「OK」をクリックしてソースを追加します。

背景は自動的に透過されます。配信画面に自然に重なって表示されます。

---

## 注意事項

- 本ツールはローカルネットワーク内で動作します。インターネット経由での配信視聴者には見えません。
- 配信中に操作画面のブラウザタブを閉じても、OBS 側の表示は最後の状態を保持します。
- 起動に失敗する場合は、Node.js が正しくインストールされているかご確認ください。
